# PolicyWell

**Building the Intelligence Layer for Insurance.**

PolicyWell is an AI-native insurance intelligence platform. Sprint 1 delivers the conversational onboarding engine, document ingestion, hybrid contextual intelligence, deterministic PolicyWell scores, agent workspace, reports, and an investor demo flow.

## Quick start

```bash
npm install
npm test
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
